## 排版约定

- **B 站专栏不完全支持 Markdown**：发布时把每篇 `.md` 内容粘贴到 B 站专栏富文本编辑器，标题与段落会被识别，但表格、`<details>` 折叠不会渲染。需要人工调整：
  - 表格 → 改成项目符号或自然段
  - `<details>` 折叠英文原文 → 改为「英文原文（节选）：...」自然段
  - 链接保留即可，B 站会自动生成可点击文本
- **图片**：每篇正文中提到的「`[配图]`」占位符，发布时手动从 [downloads/](../../downloads/) 镜像或本仓库 [uap_index.json](../../uap_index.json) 中获取原始 URL，上传到 B 站后在编辑器中插入。
- **首图**：每篇文档顶部都标记了推荐的封面图候选（缩略图远程 URL），可直接在 B 站设置封面时使用。
- **字数**：每篇约 6000–10000 字（深度长文样式）。
